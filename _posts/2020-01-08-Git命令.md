<h3>Git常用命令</h3>
* Git 常用命令及注释，不是详细教程！

>1、初始化本地 Git 仓库

  ```shell
  git init
  ```

>2、添加身份信息

  ```shell
  # 配置用户名
  git config --global user.name "Alessandro-Pang"
  # 配置邮箱
  git config --global user.email myEmail@qq.com
  ```

>3、设置 Git 文本编辑器

  ```shell
  # 文本编辑器偏好设置，这里指的是设置了emacs编辑器
  git config --global core.editor emacs
  ```

>4、查看配置信息

  ```shell
  git config --list
  ```

>5、查看 Git 版本

  ```shell
  git version
  ```

>6、查看代码库状态

  ```shell
  git status
  ```

>7、添加更新代码到缓冲区

  ```shell
  # 添加某个文件
  git add <fileName>
  # 添加所有更新的文件
  git add .
  ```

>8、提交代码

  ```shell
  git commit -m 这里是注释
  ```

>9、连接一个远程仓库

  ```shell
  git remote add origin <git-link-server>
  ```

>10、上传已经提交的代码

  ```shell
  # 首次提交需要加上 -u,之后就不再需要了
  git push -u origin master
  ```

>11、从远程仓库更新到本地

  ```shell
  git pull origin master
  ```

>12、比较代码文件的差异

  ```shell
  git diff
  ```

>13、获取标签

  ```shell
  git log
  ```

>14、插入标签

  ```shell
  git log v1.0
  ```

>15、克隆一个项目到本地

  ```shell
  git clone <git-link-server>
  ```
